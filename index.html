About this event

Are you a parent looking for summer activities that will help your kids become the best they can be? We've created our Frontend development program which aims to introduce teenagers into coding, with a focus on design thinking and creativity. This summer, your child will learn how to design mock-ups and build beautiful and appealing websites using latest tools and technologies.

Designing the web with css

Designing beautiful web UIs using figma

Develop the skills to solve design problems like a pro.
